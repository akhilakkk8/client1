This component, SelectionPreview, is a typed React functional component that accepts two props — a text string and an onSelect callback.

It maintains local state via useState to track the currently selected value. The useEffect hook fires whenever selected changes, invoking the parent's onSelect callback with the updated value.

The render output is a minimal <div> wrapper that displays the raw text prop — designed as a lightweight selection trigger for a larger AI-powered context menu.
